# python-project
## 📖 Project Overview

This project analyzes the Sample Superstore dataset using Python. The objective is to explore sales performance, profit trends, customer behavior, and regional performance through data visualization and exploratory data analysis (EDA).

---

## 🎯 Objectives

- Analyze sales and profit
- Compare sales across regions
- Identify top-performing categories
- Analyze shipping modes
- Visualize business trends
- Generate business insights

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## ✨ Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Analysis
- Profit Analysis
- Category Analysis
- Regional Analysis
- Correlation Heatmap
- Data Visualization
